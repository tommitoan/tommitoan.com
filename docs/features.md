# Features

## Gateway Homepage

- full-screen animated gateway at `/`
- 3 interactive portals: `Tech`, `Discover`, `Feng Shui`
- portal-specific background effects
- animated 3D planet per portal
- floating astronaut foreground and warp transition
- reduced-motion aware route transition timing

## Tech Route

- hero, overview, skills, experience, education, projects, contact
- structured content from `src/content/portfolio.ts`
- animated section reveals and orbit scene

## Discover Route

- route intro and route purpose
- homelab section with architecture image
- public/private service groups
- experiments and channels

## Feng Shui Route

- route overview and rationale
- Bazica live demo and repository links
- pillars, growth lanes, and guiding principles

## Platform Features

- Next.js App Router
- static export build
- sitemap, robots, manifest, and metadata helpers
- ESLint, TypeScript typecheck, and unit tests for gateway config helpers
